## OSI七层模型与TCP/IP四层模型，以及其对应的网络协议





| OSI七层模型 | TCP/IP四层模型 | 网络协议      |
| ----------- | -------------- | ------------- |
| 应用层      |                | HTTP、FTP     |
| 表示层      |                | Telnet        |
| 会话层      | 应用层         | DNS           |
| 传输层      | 传输层         | TCP、UDP      |
| 网络层      | 网络层         | IP、ICMP、ARP |
| 数据链路层  | 数据链路层     | Ethernet      |
| 物理层      |                | IEEE 802.11   |
